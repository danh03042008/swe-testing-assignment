## pytest vs unittest
The two most used Python testing frameworks are Pytest and Unittest. Unittest is an inbuilt testing library in Python, that means that it does not need any installation and it is always present. However it requires a lot of boilerplate codes, you have to write classes and invoke certain methods, and hence test more and are more difficult to read.

Pytest on the other hand is far simpler and cleaner. Tests are simply functions, syntax is very simple and error messages are far easier to comprehend.

In this project I have used Pytest since it is easy to use, less code is needed and is an industry standard in the modern python projects.
